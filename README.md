 # Parsing of Website Pokemon

 # Вашему вниманию представлено приложение "Покемоны", которое реализует парсинг данных с сайта через удобный и понятный интерфейс.

 ## Принцип работы приложения

После запуска приложения пользователь видит на экране карточки 20-ти покемонов. В каждой карточке есть изображение покемона, его имя, и ссылка на страничку сайта с описанием покемона.
Также пользователь может получить краткую справочную информацию о конкретном покемоне.

 ### Взаимодействие с пользователем через кнопки и подсказки
Для перемещения между страницами в нижней части экрана находятся 3 кнокпки:
 - ">" Осуществляет переход на следующую страницу с информацией;
- "<" Осуществляет переход на предыдущую страницу с информацией; 
- "Pokeball" Осуществляет переход на стартовую страницу с информацией;
- При наведении на любую кнопку, высвечвается подсказка с описанием функционала кнопки.
## Получение основной информации о покемоне
Пользователю необходимо нажать на картинку с покемоном, после чего на экране приложения появится краткая справка. 
Для выхода обратно в режим просмотра покемонов нажмите на кнопку "<".
